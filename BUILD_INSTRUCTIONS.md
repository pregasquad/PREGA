# How to Get Your Windows App (No Programming Required!)

Follow these simple steps to get your PREGASQUAD Manager Windows app.

## Step 1: Create a Free GitHub Account

1. Go to https://github.com
2. Click "Sign up"
3. Create your free account

## Step 2: Download This Project

1. In Replit, click the three dots menu (...) at the top left
2. Select "Download as zip"
3. Save the file to your computer

## Step 3: Upload to GitHub

1. Go to https://github.com
2. Click the "+" button at the top right
3. Select "New repository"
4. Name it: pregasquad-manager
5. Keep it "Public" (free)
6. Click "Create repository"
7. On the next page, click "uploading an existing file"
8. Extract the zip file you downloaded
9. Drag all the files into the upload area
10. Click "Commit changes"

## Step 4: Build Your App (Automatic!)

1. In your GitHub repository, click the "Actions" tab at the top
2. You'll see "Build Windows App" on the left
3. Click on it
4. Click the "Run workflow" button (on the right side)
5. Click the green "Run workflow" button
6. Wait about 10-15 minutes for the build to complete

## Step 5: Download Your App

1. After the build finishes (green checkmark), click on it
2. Scroll down to "Artifacts"
3. Click "PREGASQUAD-Manager-Windows" to download
4. Extract the zip file
5. Run the installer (.exe file)

## Done!

Your PREGASQUAD Manager is now installed on your Windows computer!

---

## Troubleshooting

**Build failed?**
- Make sure all files were uploaded correctly
- Try running the workflow again

**Can't find Actions tab?**
- Make sure you're logged into GitHub
- The tab appears after uploading files

**Need help?**
- The app works offline with its own local database
- All your salon data is stored on your computer
